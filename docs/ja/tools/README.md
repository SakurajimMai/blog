## 概要

総合ツールキットは、関連ツールの使用時にユーザーのハードルを下げることを目的としています。

## サードパーティツール

<LinkCard title="ブログカバー画像ジェネレーター" href="https://coverview.lruihao.cn/editor">

**Unsplash** の画像を使用して、高品質なブログカバー画像を生成します。

</LinkCard>

<LinkCard title="aicolors.co - AI カラースキームジェネレーター" href="https://aicolors.co/">
AIを活用して、ウェブサイトのテーマカラーや配色を生成します。
</LinkCard>

<LinkCard title="bolt.new - AI コードジェネレーター" href="https://bolt.new/">
サイトに個性的な機能を追加したいが、技術的な制限で実現できない場合に、このツールを使ってコードを素早く生成できます。
</LinkCard>

<LinkCard title="テキストからSVGへ" href="https://text-to-svg.tool.tokyo/zh">
テキストをSVG形式に変換します。アート文字やロゴの生成に適しており、アニメーション効果を付加することも可能です。
</LinkCard>

<LinkCard title="CSS 変数エディター （Chrome ブラウザ拡張機能）" href="https://www.cssvariables.com/">
CSS変数内のカラー値を管理し、サイトのUIスタイルを素早くカスタマイズするのに役立ちます。
</LinkCard>
