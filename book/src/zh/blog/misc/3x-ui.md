# 面向小白的科学上网教程(3x-ui篇)

## 服务器选购

获取一台国外的服务器

像阿里云腾讯云这种服务器就需要关闭防火墙

## 连接服务器

使用SSH工具进行连接，例如[termiux](https://termius.com/index.html)

将获取的ip，账号和密码配置后连接服务器.

## 安装3x-ui

在命令行输入

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

输入：y

获得信息

![3-2](https://s2.ixacg.com/2025/07/23/1753278389.avif)

接下来浏览器打开，输入红框中的URL，登录后进行密码配置即可
